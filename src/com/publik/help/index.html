INFORMATION MANAGEMENT SYSTEM APPLICATION HELP

This is the default help file for the INVENTORY APPLICATION  which is packaged with the application. This help information is designed to assist the user[s] of this application for their ease of use and navigation through the software.
This software package ,comes with many functionalities which are sufficient for the day to day buying and selling of products.But some extra functionalities can be unlocked if this applications is upgraded to the PRO VERSION. The table below shows the functionalities of this application.
INVENTORY APPLICATION FUNCTIONALITIES
Default Inventory Application
Functionality Description
Add Child: used to add details of a new child to the system
Add staff: used to add details of new staff members to the system
Update Details: used to supply recent information to the system of the already existing members
Reports: used to generate a written document of the data in the system for example a report of a certain child of staff member.
User Accounts: used to register new users of the system and also update user details for example their username or their level of authority to the system
