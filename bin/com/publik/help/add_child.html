ADD CHILD

This functionality is used to save new children to the system; to use this functionality click on the application icon to run the software and log in to get the home window as shown below:




{photo}
  • On the main window click on the Add child icon and a form will show up {photo}
  • Fill on all spaces available but the empty fields preceded by red stars are compulsory.
  • Use the slider to move the form up and down to add more details
  • On completion click on the save button to save the entered data
  • In case you want to erase all data entered before saving click on the reset button
